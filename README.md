# Mobile Cash Monitoring App

## Project Overview
The Mobile Cash Monitoring App is a web-based application designed to facilitate the management and tracking of employee payouts. Users can input payout details, track total payouts, and view summaries of payouts per employee, all within a clean and user-friendly interface. This application utilizes modern web technologies to ensure responsiveness and accessibility on various devices.

## Installation
To run this application locally, you need to follow these steps:

1. **Clone the repository** (you should have Git installed):
   ```bash
   git clone https://github.com/yourusername/mobile-cash-monitoring.git
   ```
2. **Change directory** into the project folder:
   ```bash
   cd mobile-cash-monitoring
   ```

3. **Open the `index.html` file** in your preferred web browser:
   ```bash
   open index.html
   ```
   or simply double-click on the `index.html` file.

## Usage
1. **Fill in the Payout Form**: Enter the payout date, employee name, and payout value in the respective fields.
2. **Submit the payout**: Click on the "Submit Payout" button to save the payout. Note that the payout date can only be set once per session.
3. **View Payout Summary**: After submission, the total payout and individual payouts by employee will be displayed.
4. **Clear payout list**: To reset the form and clear the payout list, click on the "Clear List" button.

## Features
- Input and track employee payouts with detailed summaries.
- Total payout calculation displayed in real-time.
- List of payouts aggregated by employee for easy reference.
- Responsive design utilizing Tailwind CSS.
- User confirmation for clearing the payout list.

## Dependencies
This project includes the following dependencies:
- [Tailwind CSS](https://tailwindcss.com/): For utility-first styling.
- [Font Awesome](https://fontawesome.com/): For icons.
- [Inter Font](https://rsms.me/inter/): A beautiful, variable font for better typography.

All dependencies are included via CDN links in the `index.html` file.
